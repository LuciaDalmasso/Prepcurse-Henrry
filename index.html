¿Cómo crear mi página web?

Empezando mi página web

Para comenzar, será necesario seguir una serie de pasos que nos van a brindar la base sobre la cual se va a apoyar todo el proyecto. Estos pasos incluyen desde la instalación de los programas que vamos a utilizar hasta los primeros caminos en la creación del código

1- Programas que vamos a utilizar

  • Visual Studio Code
  • Git Bash

2- Repositorio remoto

Git Hub (aquí debemos crear una cuenta)

3- Crear el entorno de trabajo

En esta etapa es necesario crear la carpeta donde vamos a guardar todos los archivos relacionados al proyecto. Para esto, en nuestra pc vamos a crear una carpeta (ésta se puede hacer de forma manual en el entorno de Windows o bien, se puede crear utilizando código desde Vs code o git bash, para ello ver los primeros videos en el prep curse de Henrry). Esta carpeta que hemos creado puede ser utlizada de manera remota sin necesidad de enviar los archivos para su edición, y para ello es que sirve la cuenta en Git Hub, donde vamos a crear un nuevo repositorio y vamos a vincular nuestra carpeta desde nuestro almacenamiento local. Para ello vamos a convertirla en repositorio utilizando el Vs code. Desde una nueva terminal vamos a vincular la carpeta con el repositorio remoto de Github con el siguiente esquema de código:

  1. cd (aquí se coloca la dirección o nombre de la carpeta donde vamos a trabajar para ingresar en ella)
  2. Touch "nombre.html" (vamos a crear nuestro archivo html)
  3. Git init (para convertir la carpeta creada en repositorio). En esta etapa ya se puede crear un commit, es decir una instantánea del estado en el cual se encuetra el proyecto hasta ese momento, utilizando el código git add "nombre del archivo" o git add . para hacerlo en todos los archivos de la carpeta Luego se utiliza el código git commit -m y algun texto de referencia "entre comillas"
  4. git remote add origin "direccion del repositorio remoto" (debemos copiar esta direción en la sección "code" del repositorio que creamos en Github)
  5. git branch -m main
  6. git push -u origin main

Con estos pasos ya tenemos nuestro archivo html creado y guardado en un repositorio que va llevar un seguimiento por parte de git, a demás lo vamos a poder guardar de manera remota en la cuenta de Github

4- Estructura de la página web

Abrir el Vs code, arrastramos la carpeta repositorio hacia el entorno de trabajo del vs y desde ahi ya podemos ver todos los archivos que contiene esta carpeta, que por el momento solo es el archivo html que creamos.

Abrir el archivo html y en la columna central del vs code ya se puede comenzar a editar. Ahi mismo vamos a copiar la estructura base para nuestra web:

< !DOCTYPE html>

< html lang="es">

< head>

< title>Título

< /head>

< body>

< h1>Título< /h1>

< p>Texto< /p>

< /body>

< /html>

Etiquetas

Dentro de este código madre vamos a poder darle forma a nuestra página web volcando toda la información y multimedia necesaria y para ello vamos a tener algunas herramientas que nos da vs code, una de ellas son las etiquetas. Cada etiqueta tiene una función específica con la que se puede programar el formato o uso de algun elemeto de la página web, ya sea para el título, párrafos de texto, listas de elementos, etc.

Etiquetas h

Las etiquetas < h > y < / h > nos permiten establecer títulos o secciones con mayor jerarquía en la web y estas van del 1 al 6

Etiquetas p

Mientras que las etiquetas < p > y < / p > nos permiten agregar párrafos de texto. La sintáxis correcta indica que los caracteres no deben estar separados, pero para poder ponerlo en exeplo se excribió de esta manera

Etiquetas ol y li

para crear un listado de elementos ORDENADOS, lo vemos a hacer con las etiquetas < ol > y < /ol > que son listas ordenadas, pero que a tu vez dentro llevan las etiquetas li que son list item

  1. Item 1
  2. Item 2
  3. item 3

Etiquetas ul y li

Mientras que, para crear un listado de elementos NO ORDENADOS, lo vamos a hacer con las etiquetas < ul > y < /ul > y como en la lista anterior, también se deben colocar los items con las etiquetas < li > y < /li >

  • Item 1
  • Item 2
  • item 3

Separadores

Para separar secciones en el texto se pueden usar las etiquetas < hr > que van a crear separadores lineales

Etiquetas span

Para colocar textos cortos y que queden uno al lado de otro se usan las etiquetas < span > y < /span >

Por ejemplo: Home Noticias Clima

LISTADO DE LAS ETIQUETAS MÁS UTILIZADAS

  • < html>: Raíz de un documento HTML, engloba todo el contenido de la página.
  • < head>: Contiene metadatos/información para el documento, como el titulo y enlaces a scripts y hojas de estilo.
  • < title>: Define el título del documento, que se muestra en la barra de titulo del navegador o en las pestañas.
  • < body>: Representa el contenido principal de un documento HTML, visible para el usuario.
  • < h1>, < h2>,< h3>, < h4>, < h5>, < h6>: Son utilizadas para definir encabezados, siendo < h1> el más Importante y < h6> el menos.
  • < img>: Incrusta una imagen en el documento.
  • < div>: Define una división o sección en un documento, generalmente utilizado como contenedor para diseño o estilizado.
  • < span>: Sirve para agrupar elementos en línea para aplicarles estilo o por razones de script. < ul>: Define una lista desordenada (con viñetas).
  • < ol>: Define una lista ordenada (con números).
  • < li>: Define un item de lista que debe estar dentro de < ul>, < ol>, o < menu>.
  • < br>: Inserta un salto de linea simple.
  • < table>: Define una tabla.
  • < tr>: Define una fila en una tabla.
  • < td>: Define una celda en una tabla.
  • < th>: Define una celda de encabezado en una tabla.
  • < form>: Define un formulario HTML para la entrada del usuario. < input>: Define un campo de entrada dentro de un formulario.
  • < label>: Define una etiqueta para un elemento < input>.
  • < button>: Define un botón clickeable.
  • < select>: Define un menú desplegable.
  • < option>: Define una opción dentro de un < select>.
  • < textarea>: Define un campo de texto multilinea.
  • < link>: Utilizado para enlazar hojas de estilo externas.
  • < meta>: Define metadatos que no pueden ser representados por otros elementos HTML.
  • < script>: Se utiliza para insertar o hacer referencia a un script ejecutable.

Estilos CSS

Para darle estilo, color y mejorar la estetica de nuestra web es necesario utilizar el código CSS. Este código se va a trabajar en un archivo distinto a nuestro .html original y, por lo tanto, va a ser necesario vincular ambos archivos. A continuaión los pasos a seguir.

  1. Desde la columna explorer en VS CODE vamos a crear un nuevo archivo, el cual llevará la extensión .CSS
  2. Para vincularlo con nuestro .html, dentro de éste ultimo y a continuación de la etiqueta head de nuestro código base, se debe copiar el siguiente código: < link rel="stylesheet" href="./estilos.css" / >

Con estos dos sencillos pasos, hemos vinculado nuestro código html con el css y de esta manera, todos los estilos que definamos en el archivo css serán aplicados en nuestra web
